01

¿Dónde entran los grupos de edad?

El grupo no es el foco del estudio — es una variable de control que permite limpiar los efectos del diseño de tarea

Dos tipos de variables, dos funciones distintas

🎯 Variables de diseño — LO QUE ESTUDIAMOS

  • Espacio (tamaño de campo)
  • Agrupación (nº jugadores)
  • Polaridad (con/sin porterías)
  • Equilibrio (igualdad numérica)

"¿Cómo afecta cada variable a las demandas?"

🔧 Grupo + Jugador — LO QUE CONTROLAMOS

  • Grupo (Infantil-Cadete, Juveniles, Senior, Neskak)
  • JugadorID (identidad individual)

"Absorben el ruido para que los efectos del diseño sean limpios"

Un Senior y un Infantil en la misma tarea producen valores diferentes simplemente por ser poblaciones distintas. Si no se controla, los efectos de la tarea se confunden con las diferencias entre grupos.
¿Cómo se controla?
G
Grupo como efecto fijo Ajusta las diferencias basales entre poblaciones
J
Jugador como efecto aleatorio Corrige la pseudoreplicación (mismo jugador, múltiples tareas)
02

¿Para qué es el EDA?

Cada análisis exploratorio toma una decisión concreta que condiciona el modelado posterior

5 preguntas que el EDA responde

📊 ¿Qué distribución tienen las VDs?

Histogramas, QQ-plots, ratio varianza/media → Decide la familia del GLMM (Binomial Negativa, Gaussiana, Gamma)

🔍 ¿Hay problemas de calidad?

Missings, duplicados, outliers, inconsistencias → Produce el dataset limpio final

👥 ¿Cuánta variabilidad es entre jugadores?

ICC preliminar → Justifica el efecto aleatorio por jugador en el GLMM

📈 ¿Los efectos son paralelos entre grupos?

Gráficos de interacción Grupo × Diseño → Justifica el modelo con/sin interacciones

🔗 ¿Hay interacciones entre variables de diseño?

Gráficos de interacción Diseño × Diseño → Decide si Espacio × Agrupación (espacio individual de juego) entra en el modelo principal

ANOVA — Exploración

  • ✓ Ranking rápido de importancia (η²p)
  • ✓ Primera impresión de efectos
  • ✓ Documenta violaciones de supuestos
  • ✓ Línea base para comparar con GLMM
  • ✗ No corrige pseudoreplicación
  • ✗ Asume normalidad y homocedasticidad
→

GLMM — Análisis Principal

  • ✓ Modela distribución real de los datos
  • ✓ Efecto aleatorio por jugador (ICC)
  • ✓ Interacciones Grupo × Diseño
  • ✓ EMMs generales y por grupo
  • ✓ Errores estándar honestos
  • ✓ Rate Ratios interpretables

Si ANOVA y GLMM coinciden → robustez. Si difieren → el GLMM es más fiable y se explica por qué.

04

¿Cómo funciona el GLMM?

El modelo descompone cada observación en componentes: diseño + grupo + jugador + residuo

Estructura del modelo

log(toques/min) = β₁·Espacio + β₂·Agrupación + β₃·Polaridad + β₄·Equilibrio + β₅·Esp×Agr
+ β₆·Grupo + Grupo×Espacio + Grupo×Agrupación + ...
+ u_jugador + residuo
ComponenteTipoFunción
Espacio, Agrupación, Polaridad, EquilibrioEfecto fijoLo que estudiamos — efectos del diseño
Espacio × AgrupaciónEfecto fijoInteracción de diseño (IPA)
GrupoEfecto fijoControl — diferencias basales
Grupo × cada VI diseñoEfecto fijoPermite desglosar efectos por grupo
JugadorIDAleatorioControl — variabilidad individual

Familia distribucional decidida por el EDA: Binomial Negativa (conteos técnicos), Gaussiana o Gamma (variables físicas)

"El espacio reducido incrementó los toques/min un 28% de media. Este efecto fue consistente entre grupos, oscilando entre +22% en Neskak y +35% en Infantil-Cadete."

¿El desglose es significativo?

La interacción Grupo × Espacio se testea formalmente:

  • No significativa (p > .05): El efecto general es representativo. Desglose es descriptivo.
  • Significativa (p ≤ .05): Los grupos responden de forma diferente. Hallazgo relevante.

En ambos casos, se presentan las dos tablas — el test dice si la variación entre grupos es ruido o señal.
